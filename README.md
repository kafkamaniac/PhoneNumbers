# 📞 Corporate Phone Directory

Веб-приложение телефонного справочника предприятия с разграничением прав доступа, поиском, пагинацией и администрированием записей.

---

## 🧰 Технологии

**Backend:**
- C# / .NET
- ASP.NET Core
- Entity Framework Core
- MySQL
- Логирование

**Frontend:**
- React (Vite)
- TypeScript

---

## 📌 Описание

Проект представляет собой внутренний телефонный справочник компании, предназначенный для хранения и быстрого поиска контактной информации сотрудников.

---

## 📇 Структура данных

Каждая запись содержит:

- ФИО  
- Должность  
- Подразделение  
- Корпус  
- Кабинет  
- Внутренний телефон  
- Городской телефон  
- Мобильный телефон  
- Email  

---

## 👥 Роли пользователей

### Пользователь
- Просмотр списка контактов
- Поиск по справочнику
- Использование пагинации

### Администратор
- Добавление новых записей
- Редактирование контактов
- Удаление записей

---

## 🔍 Функционал

- Поиск по выбранному полю (ФИО, должность, подразделение и т.д.)
- Пагинация (5 / 10 / 20 / 50 записей на страницу)
- Табличное отображение данных
- Обновление данных через API

---

## ✏️ Администрирование

Администратор может:
- Создавать новые записи
- Редактировать существующие
- Удалять записи

Все изменения сохраняются в MySQL через backend.

---

## 🔐 Безопасность

- Хэширование паролей с солью
- Разделение прав доступа (User / Admin)
- Ограничение попыток входа (защита от перебора пароля)
- Контроль доступа к административным операциям

---

## 🧾 Дополнительно

- Логирование действий пользователей
- Резервное копирование базы данных
- Работа через REST API

---

## 🗄 Архитектура

Frontend (React + Vite + TypeScript)  
→ REST API  
Backend (ASP.NET Core + EF Core)  
→ MySQL

---

## ⚙️ Запуск проекта

### Backend
dotnet restore
dotnet run
### Frontend
npm install
npm run dev

🗃 Миграции базы данных
dotnet ef migrations add InitialCreate
dotnet ef database update
